# Calculadora de Costes de Impresión 3D

Una herramienta web profesional diseñada para makers y negocios de impresión 3D. Calcula el coste real de tus piezas y genera presupuestos en PDF al instante.

## Acceso a la Herramienta

Puedes utilizar la calculadora ahora mismo desde tu navegador, sin necesidad de instalar nada:

 * [CLICK AQUÍ PARA ABRIR LA CALCULADORA](https://anxovc.github.io/Calculadora3D/)



##  Características

Esta aplicación te ayuda a no perder dinero calculando costes ocultos que a menudo se ignoran.

* **Coste de Material:** Cálculo preciso basado en el peso, precio de la bobina y porcentaje de merma (desperdicio).
* **Costes Operativos (Opcionales):**
    * Electricidad (basado en consumo W y precio kWh).
    * Mano de obra (tiempo de preparación y post-procesado).
    * Amortización de la máquina (desgaste por horas de uso).
* **Generador de Facturas PDF:** Crea un documento formal tamaño A4 con el desglose de costes, listo para enviar al cliente.
* **Interfaz Limpia:** Diseño por pasos desplegables para mantener el orden.

## Tecnologías Utilizadas

* **HTML5 & CSS3:** Diseño moderno y adaptable.
* **JavaScript:** Motores de cálculo en tiempo real.
* **html2pdf.js:** Librería para la generación de documentos PDF en el navegador.

## Instalación Local (Opcional)

Si prefieres descargar el código para modificarlo:

1.  Clona este repositorio:
    ```bash
    git clone [https://github.com/anxovc/Calculadora3D.git](https://github.com/anxovc/Calculadora3D.git)
    ```
2.  Abre el archivo `index.html` en tu navegador.
